# 分诊标签

技能以五个规范分诊角色说话。本文件把这些角色映射到本 repo issue 跟踪器实际使用的标签字符串。

| mattpocock/skills 中的标签 | 本跟踪器的标签 | 含义 |
| --- | --- | --- |
| `needs-triage` | `needs-triage` | 维护者需要评估此 issue |
| `needs-info` | `needs-info` | 等待报告人补充信息 |
| `ready-for-agent` | `ready-for-agent` | 已完整规格化，可供 AFK 代理接手 |
| `ready-for-human` | `ready-for-human` | 需要人工实现 |
| `wontfix` | `wontfix` | 不会处理 |

当技能提到某个角色（如「打上 AFK-ready 分诊标签」），使用本表中对应的标签字符串。

编辑右栏以匹配你实际使用的词汇。
