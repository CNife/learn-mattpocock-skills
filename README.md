# learn-mattpocock-skills

[![skills.sh](https://skills.sh/b/mattpocock/skills)](https://skills.sh/mattpocock/skills)

Matt Pocock [Skills](https://github.com/mattpocock/skills) 的中英双语学习图谱与对照阅读站。

将 Matt Pocock 的 AI Agent 技能（grill、domain-modeling、prototype 等）转化为双栏对照 HTML 页面，方便中文开发者理解和学习。

在线阅读：[https://cnife.github.io/learn-mattpocock-skills](https://cnife.github.io/learn-mattpocock-skills)

HTML 排版基于 [tw93/Kami](https://github.com/tw93/Kami) 制作。
